Considerações para testar a versão de desenvolvimento:

A tela de menu principal funciona tanto com controle (analógico e A) quanto com o mouse (não funciona com teclado)
A tela de seleção de personagens funciona com controle (os botões aparecem na tela) quanto com o teclado (não funciona com o mouse)
A tela de extras funciona com controle e mouse (as abas funcionam com botões LB e RB).
HACK PARA DESBLOQUEAR TODOS O CONTEÚDO EXTRA: na tela de extras, apertar a tecla + do key pad da direita seguido de enter.
A cena de jogo (após selecionar um personagem) funciona tanto com teclado quanto com controle.

Mapeamentos pra tela de seleção de personagens:
	A = Espaço
	B = Esc
	X = Alt esquerdo
	Start = Enter

Para testar o projeto, basta rodar a build.
Se quiser testar as cenas individuais do jogo, é necessário começar ou pela cena de main menu ou pela cena de character select (não funciona se começar pela cena do jogo)
Isso se deve ao fato de precisar de referências passadas da cena de seleção de personagens.
